# Gnuvil
소프트웨어 중앙 동아리(그누빌)


## 🖥️ 활동 내용
- 빅데이터(데이터사이언스) 스터디 활동<br>
- 강의와 교재를 활용하여 빅데이터 분석에 대한 내용을 익히고, 매주 실습과 약 2시간의 토론을 통해 통계적 지식과 빅데이터 분석법을 익힘
<br>

* 2022-2학기 : 22/09/01 ~ 22/12/21
